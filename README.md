Задание 2 для самостоятельного выполнения
Указание. Создать класс выполняющий задание для одного каталога.
Поиск файлов реализовать с помощью функций FindFirstFile() и
FindNextFile(). Главный поток обеспечивает реакцию формы.
Вспомогательный поток запускает потоки для поиска файлов и ожидает от
них результатов (использует для этого средства синхронизации). 

18.Заданы два каталога. Для каждого из них подсчитать количество
каталогов, содержащих более чем m файлов. Сравнить их количество.
